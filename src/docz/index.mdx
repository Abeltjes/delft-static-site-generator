# Static Stite Generator documentation

## Installation
To install the development environment you will require `node@8.11.x` and `yarn` installed.

Gatsby needs to be installed globally:

```sh
npm install -g gatsby
```

To install run: 
```sh
yarn
```

To start the main application run: 

```sh
gatsby develop
```

and your server will be available at [http://localhost:8000/](http://localhost:8000/)

To start this documentation application, run: 

```sh
yarn docz:dev
```

and the documentation will be available at [http://localhost:5001/](http://localhost:5001/)
